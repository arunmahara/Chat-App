# Chat App
A real-time chat application made with Django.

## Features:
Create Chat Room <br>
Join Chat Room <br>
Chat Backup <br>
Real-Time Chat With Different Users <br>

## Making Use Of:
Django Channels <br>
Channel Redis <br>
Generic Websocket Consumer <br>
